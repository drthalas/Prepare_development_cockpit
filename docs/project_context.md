# Project Context

## Project

Prepare Development Cockpit is intended to become a workspace for preparing and coordinating development work before implementation starts.

## Phase 0 Objective

Phase 0 creates the local and GitHub-ready foundation:

- Next.js App Router project.
- TypeScript configuration.
- Basic documentation.
- Environment example.
- Health endpoint.
- Git repository and GitHub remote.

## Current Scope

This phase intentionally avoids product features and external service setup. The repository should be ready for future tasks PDC-001 through PDC-003 without introducing premature infrastructure.

## Out of Scope

- Railway project creation.
- Railway services.
- Real AI generation.
- Database models.
- Authentication.
- Billing.
- Linear API integration.
- Production deployment.
